CSS Flexbox - A Quick Reference

flex-direction:
row | row-reverse | column | column-reverse

row

.flex__parent {
    display: flex;
    flex-direction: row;
}

row reverse

.flex__parent {
    display: flex;
    flex-direction: row-reverse;
}

column

.flex__parent {
    display: flex;
    flex-direction: column;
}

column reverse

.flex__parent {
    display: flex;
    flex-direction: column-reverse;
}

flex-wrap:
nowrap | wrap | wrap-reverse | inherit

will affect flex-items with a "width".

nowrap

.flex__parent {
    display: flex;
    flex-direction: row;
    flex-wrap: nowrap;
}

wrap

.flex__parent {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

wrap-reverse

.flex__parent {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap-reverse;
}

align-items:
flex-start | flex-end | center | baseline | stretch

changes invisible unless flex-container has a "height".
Exception: baseline.
